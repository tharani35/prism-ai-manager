# 🚀 PRISM AI Manager

**AI-Powered Content Management Platform**

PRISM AI Manager is a modern AI-driven content management platform that helps users generate, organize, manage, and analyze content using artificial intelligence. The platform combines a premium user experience with powerful content creation tools, analytics, authentication, and content management workflows.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/tharani35/prism-ai-manager.git
cd prism-ai-manager
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file:

```env
MONGODB_URI=your_mongodb_connection_string
OPENAI_API_KEY=your_openai_api_key
JWT_SECRET=your_secret_key
```

### Start Development Server

```bash
npm run dev
```

### Build Production Version

```bash
npm run build
```
